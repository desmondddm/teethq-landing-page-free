# Teeth Q — Homepage Redesign (Free)

A complimentary homepage redesign for **Teeth Q Dental Surgeons**, built by Desmond de Moussac.
This page is yours to keep and use, free of charge — no strings attached.

## What's included

- `index.html` — the complete redesigned homepage (single file, no build step)
- `assets/` — images and logo used by the page

## How to view it

Open `index.html` in any web browser. That's it — everything is self-contained.

## How to publish it

The page is a static site, so it can be hosted anywhere:

- **Drag-and-drop:** upload the folder to Netlify Drop, Vercel, or Cloudflare Pages
- **Your existing host:** replace your current homepage with `index.html` and the `assets/` folder
- **GitHub Pages:** enable Pages on this repo (Settings → Pages → deploy from `main`)

## A note on compliance

This redesign was built to align with Singapore's healthcare advertising rules — the
**Healthcare Services (Advertisement) Regulations 2021 (HCSA)** and the
**Singapore Dental Council Ethical Code & Ethical Guidelines**. It deliberately avoids
patient reviews/ratings, popularity awards, superlatives, comparative claims, and
promotional pricing on elective treatments. It leads instead with permitted factual
signals: your specialists, services, branch network, credentials, and official
accreditations (MOH-licensed, CHAS, MediSave).
